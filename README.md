Minimal [Giter8][g8] template for Scala projects

Use `sbt new augustnagro/s.g8` to create a minimal template with:

* [Scala][scala] (2.12.0)
* [SBT][sbt] (>= 0.13.13)
* [ScalaCheck][scheck] (latest release)

Reasoning for the template's settings are detailed [here][post]
